# (통합) PyOllaMx UI 연동 가이드

이 문서는 사용자가 시각적인 인터페이스(UI)를 통해 MLX 모델과 채팅할 수 있도록 `PyOllaMx` 앱을 연동하는 방법을 안내합니다.

---

## 1. 기획 (Planning) - [본질가] 주도
- **목표**: 터미널 기반의 `mlx-lm` 서버를 `PyOllaMx`와 연결하여 사용자 친화적인 채팅 환경 구축.
- **핵심 설정**: API 엔드포인트 `http://127.0.0.1:11435`를 앱에 등록.

---

## 2. 분석 및 설정 (Analysis & Design)
- **앱 정보**: `PyOllaMx`는 Ollama 및 MLX 모델을 지원하는 macOS용 챗봇 앱입니다.
- **연동 주소**: `http://127.0.0.1:11435` (OpenAI API 호환)

---

## 3. 구현 (Implementation) - [실천가] 독재
**[실천가]**: "말 개길어. 아래 순서대로 바로 세팅하셈."

### 📥 1단계: PyOllaMx 설치 (없를 경우)
앱이 아직 없다면 아래 링크에서 최신 `.dmg` 파일을 받아 설치하세요.
- **다운로드**: [PyOllaMx GitHub Releases](https://github.com/kspviswa/PyOllaMx/releases)

### ⚙️ 2단계: 연동 설정
1. **PyOllaMx 앱 실행**.
2. 상단 메뉴 또는 설정(Settings) 아이콘 클릭.
3. **MLX / OpenAI Server** 섹션 탐색.
4. **Server URL** 칸에 아래 주소 입력:
   - `http://127.0.0.1:11435`
5. **Model Name** 입력 (필요 시):
   - `mlx-community/quantized-gemma-2b`
6. 저장(Save) 후 메인 화면으로 돌아가기.

### 💬 3단계: 채팅 시작
1. 모델 선택 드롭다운에서 **MLX** 또는 직접 입력한 모델명 선택.
2. 프롬프트를 입력하고 채팅을 즐기세요!
