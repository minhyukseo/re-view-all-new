# 영상뷰어 모드(Video Viewer Mode) 통합 요약

## [개요]
사용자로부터 URL을 입력받아 영상 컨텐츠만 추출하여 보여주는 '영상뷰어 모드'를 기획, 분석, 설계, 구현, 검수하였습니다.

## [주요 특징]
1. **정밀 파싱 API**: 유튜브, 스트리머블, 커뮤니티(개드립, 디시 등)의 영상 소스를 서버 측에서 추출.
2. **시네마틱 UI**: 어두운 배경과 앰비언트 라이트 효과가 적용된 고해상도 플레이어 페이지.
3. **글래스모피즘**: 현대적인 디자인 가이드를 준수하는 프리미엄 UI.

## [구현 파일 목록]
- `server/src/index.ts`: `/api/video-extract` 엔드포인트 추가.
- `web/app/video/page.tsx`: 영상뷰어 메인 페이지.
- `web/components/VideoPlayer.tsx`: 다목적 영상 플레이어 컴포넌트.

## [검증 결과]
- 유튜브 및 일반 MP4 소스 추출 정상 작동 확인.
- 다크 모드 레이아웃의 시각적 완성도가 높음.
